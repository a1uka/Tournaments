# Структура

# Бэкенд
* Запрос каждые 5 секунд результатов с api, последующая запись матча в БД, если его там нет

* Создание турира, указываение его формата и подробных данных

* Изменение турнира

* Удаление турнира

* Старт турнира, выбор матчей, которые надо добавить

* Турнир завершается автоматически, когда завершаются все этапы

* Этап завершается тогда, когда завершены все матчи. Этап возвращает уже готовую таблицу местами команд

* матч завершается тогда, когда для него нашлись результаты в БД.
